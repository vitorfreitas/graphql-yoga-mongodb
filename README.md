# GraphQL + MongoDB                                                                                                                             
 
Playground de desenvolvimento e aprendizado de GraphQL com banco de dados MongoDB.
 
### Scripts úteis
 
`yarn start:dev`  
 
Executa a versão de desenvolvimento com hot reload da API
